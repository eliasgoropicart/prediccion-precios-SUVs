# prediccion-precios-SUVs
Modelado predictivo de precios para SUVs en el mercado argentino. Implementación de un pipeline de ETL para +18k registros y desarrollo de un modelo de ensamble híbrido (XGBoost + MLP/Neural Networks). El proyecto aborda feature engineering avanzado y optimización de hiperparámetros, logrando un $R^2$ de 0.91 en la estimación de activos.
